# Интернет-магазин китайского чая

Техническое предложение, рисёч регуляторики и архитектура бизнес-процессов для интернет-магазина китайского чая (150–200 SKU, квиз-сегментация, e-commerce с нуля).

## Содержимое

| Файл | Что это |
|---|---|
| `proposal.html` | Техническое предложение (исходник, рендерится в PDF). Стек: Битрикс+Аспро Next · RetailCRM · МойСклад. Бюджет под ключ 1.2 млн ₽ |
| `proposal.pdf` | Печатная версия предложения |
| `architecture.excalidraw` | Диаграмма архитектуры бизнес-процессов: приёмка → фасовка (развес + свои пачки) → склад → витрина → заказ → возврат. Открыть на [excalidraw.com](https://excalidraw.com) |
| `research.md` | Рисёч: маркировка Честный знак, бронь товара, роли кладовщик/менеджер, наборы, возвраты, 12 упущенных пунктов |

## Стек предложения

- **Витрина:** Битрикс + Аспро Next (SSR, schema.org)
- **Заказ/клиенты:** RetailCRM (воронки, сегменты, рассылки, лояльность)
- **Склад:** МойСклад (партионный учёт, сроки годности, фасовка)
- **Платежи:** ЮKassa (карты+СБП) + АТОЛ Онлайн (54-ФЗ)
- **Маркировка:** Честный знак / ЭДО / ОФД (обязательна для чая с 01.04.2026)
- **Доставка:** СДЭК / Boxberry / Почта / Яндекс.Доставка

## Статус

Pre-разработка: предложение + рисёч + диаграмма. Кода магазина пока нет.
